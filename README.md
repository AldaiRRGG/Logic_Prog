# Computador Adivina el Número

## Descripción

Este proyecto consiste en un programa desarrollado en Python donde el usuario piensa un número entre 0 y 50 y la computadora intenta adivinarlo mediante las respuestas proporcionadas por el usuario.


## Funcionamiento

1. El usuario piensa un número entre 0 y 50.
2. La computadora realiza un intento para adivinar el número.
3. El usuario responde:

   * mayor
   * menor
   * correcto
4. La computadora utiliza la respuesta para realizar un nuevo intento.
5. El proceso se repite hasta adivinar el número correcto.


## Estructuras Utilizadas

* Variables
* Condicionales (if, elif, else)
* Bucles (while)
* Entrada y salida de datos (input y print)
